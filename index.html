このサイトについて

このサイトは、 HTML 、 CSS 、 javascript を使って作られたものです。
スマートフォンにも対応していますが、PCでの閲覧をおすすめします。

私は若者で、普段ゲームをプレイしていますが、その裏で何がどのように動いているのか、 どのような設計がされているのかに非常に興味を持ちました。 そのため、空いている時間には自分で何かを作ってみたいと思うようになりました。 ゲーム制作というとデザインやフロントエンドのイメージが強いかもしれませんが、実際には約80%が設計とコーディングです。 以下は、私が自由時間に作成したゲームです。 GODOTエンジン を使ってゲームを制作しており、とても楽しい時間を過ごしています。GODOTエンジンでは GDScript というプログラミング言語を使用しますが、 Python とほぼ同じ文法です。 ゲーム用のスプライト（画像）は、 Aseprite というエディターソフトでピクセルアートとして作成・修正しました。
